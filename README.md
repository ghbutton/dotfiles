# Personal dotfile
## To install
- $ git clone git@github.com:ghbutton/dotfiles.git
- $ cd dotfiles
- $ ./setup.sh
- $ # Setup nvim https://vi.stackexchange.com/questions/12794/how-to-share-config-between-vim-and-neovim
- $ # Setup Vundle see https://github.com/VundleVim/Vundle.vim
